# NLP_by_NN — Sentiment Analysis using Neural Networks

## Project Overview
https://nlp-by-nn.streamlit.app/
This project is a Neural Network based NLP system for **Sentiment Analysis**.  
The goal is to classify product reviews into three sentiment classes:

- Negative
- Neutral
- Positive

The system is not only a simple text classifier. It supports:

- Single review prediction
- Batch CSV prediction
- Sentiment probabilities
- Confidence score
- Sentiment distribution summary
- Overall dataset mood detection
- Downloadable prediction results

The project follows the Neural Networks project requirements by using Sentiment Analysis with Bag of Words / TF-IDF and an MLP model. It also includes preprocessing, train/test splitting, training monitoring, evaluation, experiments, and a GitHub submission structure.
